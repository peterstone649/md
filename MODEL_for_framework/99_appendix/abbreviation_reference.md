# Abbreviation Reference (ABBREV_REF_MFW)
 
> **Quick Reference Guide for Framework Abbreviations**

## 1. Core Prefix Abbreviations

| Prefix | Full Term | Example |
|--------|-----------|---------|
| **mfw** | Model Framework | `mfw_id`, `mfw_version` |
| **fw** | Framework | `fw_id`, `fw_version` |
| **model** | Model | `model_name`, `model_type` |
| **tpl** | Template | `tpl_id`, `tpl_category` |
| **con** | Convention | `con_id`, `con_status` |
| **val** | Validation | `val_score`, `val_result` |
| **sys** | System | `sys_name`, `sys_version` |
| **evo** | Evolution | `evo_phase`, `evo_status` |
| **axiom** | Axiom | `axiom_id`, `axi_type` |
| **integ** | Integration | `integ_type`, `integ_status` |
| **prop** | Property | `prop_name`, `prop_value` |
| **term** | Term | `term_id`, `term_name` |
| **princ** | Principle | `princ_id`, `princ_reuse`, `PRINC_MFR_REUSE` |
| **def** | Definition | `def_id`, `def_text` |

## 2. Status Abbreviations

| Status | Meaning |
|--------|---------|
| **DRAFT** | Initial creation, pending review |
| **OPEN** | Task identified, not started |
| **IN PROGRESS** | Active work being conducted |
| **PAUSED** | Work temporarily halted |
| **REVIEW** | Pending validation/approval |
| **DONE** | All work complete and validated |
| **ARCHIVED** | Historical reference |
